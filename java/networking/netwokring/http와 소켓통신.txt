http는 하이퍼 텍스트 html을 전송하는 프로토콜이다.
현재는 html뿐만 아니라 json, image 등 다양한 파일을 전송한다.
요청과 응답을 통해 통신한다.
클라이언트가 요청하면 서버가 응답한다.

소켓은 두 프로그램이 서로 데이터를 주고 받는 통신 단자이다.
서버와 클라이언트가 양방향 연결이 이루어진 통신으로 클라이언트, 서버 모두 요청과 응답이 가능하다.
실시간으로 데이터를 주고 받아야 하는 경우 자주 사용된다.
하지만 http에 비해 많은 리소스를 소모한다.
자주 데이터를 주고 받는 경우가 아닐 경우에는 http를 사용하는 것이 좋다.

http는 클라이어트->서버
소켓은 클라이언트<->서버
